# ToDo With MVC

Projeto com objetivo de aprendizado e conhecimento no desenvolvimento de um sistema em MVC de ToDo utilizando as tecnicas de Identity e Multi Usuarios

## Tecnologias

- Csharp
- Asp Net MVC
- Entity Framework (In Memory/PostGres)
- Identity
- Multi Usuarios
